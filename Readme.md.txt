
# Cronograma Churrasco

Atividade realizada para afins de estudo PUC Campinas Eng. Software.

## Descrição do Cronograma

Cronograma de churrasco foi utilizada a técnia de refinamento sucessivos, partindo das atividade principais para o desenvolvimento do projeto, gradativa e sucessivamente. ( Top-Down )
## Objetivo do Cronograma 

- Churrasco
    - Planejamento
        - Definir Quantidade de pessoas
        - Definir Local
        - Definir Quantidade de Carne
        - Definir Quantidade de bebida
    - Compras
        - Comprar Carne
        - Comprar bebida
        - Comprar restante
    - Pré Churrasco 
        - Preparar Carne
        - Gelar bebida
        - Arrumar Local
    - Churrasco
        - Assar Carne
        - Servir Carne
        - Servir bebida
